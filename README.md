## Local Setup:
 1. Uncomment these lines in app.py 
 `nltk.download('stopwords')`  
 `nltk.download('averaged_perceptron_tagger') ` 
 `nltk.download('brown')`
 2. Run *app.py* with `python app.py`.
 3. The chat will be open at [http://localhost:5000/](http://localhost:5000/)
